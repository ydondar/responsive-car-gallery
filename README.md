# Responsive Car Gallery Website

A responsive multi-page car gallery website built using HTML, CSS, and Vanilla JavaScript.

## Features

- Responsive design
- Car search functionality
- Contact form validation
- Hover animations and transitions
- Car filtering system
- Multi-page website structure
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Pages

- Home
- About Us
- References
- Cars
- Contact

## Responsive Design

Optimized for:

- Desktop
- Tablet
- Mobile devices
